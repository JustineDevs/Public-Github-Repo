# GitHub Proxy Tools 
## 🚀 GitHub Repository Transformers
Community-driven projects that turn a standard GitHub repo into specialized views.

| Tool | Domain Swap URL Pattern | Best Used For... |
|---|---|---|
| ExplainGitHub | ://explaingithub.com | AI Explanations: Getting a plain-English walkthrough of complex code. |
| GitMCP | gitmcp.io/user/repo | AI Agents: Turning a repo into a "Model Context Protocol" source for tools like Claude/Cursor. |
| GitHub1s | ://github1s.com | Web Editor: Opening the code in a full VS Code interface directly in the browser. |
| Git History | github.githistory.xyz/user/repo | Visual Timeline: Scrubbing through the history of a file with a smooth, interactive UI. |
| DeepWiki | ://deepwiki.com | Deep Learning Docs: Transforming repositories into structured, readable wikis (often AI-enhanced). |

------------------------------

### 🛠️ GitHub Enhancement Tools- [ExplainGitHub](https://explaingithub.com/) - AI-powered codebase explanations.
- [GitMCP](https://gitmcp.io/) - Repository to AI Context (MCP) transformer.
- [GitHub1s](https://github1s.com/) - Instant VS Code browser view.
- [Git History](https://githistory.xyz) - Beautiful visual file history.
- [DeepWiki](https://deepwiki.com/) - Knowledge-base and wiki generator for repos.
